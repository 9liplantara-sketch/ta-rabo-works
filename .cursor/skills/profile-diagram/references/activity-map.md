- **tier 0（核）**: マテリアルデザイン / 空間デザイン（2中心）

## ⚠️ screen ブレンドの実装ルール

**ラジアルグラジェントで端をα=0にしてはいけない。**
端がα=0になると screen 合成の計算が崩れ、重なり部分の色が正しくならない。

### 正しい実装

```js
ctx.save();
ctx.globalCompositeOperation = 'screen';
ctx.beginPath();
ctx.arc(x, y, r, 0, Math.PI * 2);
ctx.fillStyle = `rgba(${c[0]},${c[1]},${c[2]}, 0.65)`;
ctx.fill();
ctx.restore();
```

### screen 合成の結果早見表

| 重なり | 結果 |
|---|---|
| 赤 + 緑 | 黄 |
| 赤 + 青 | マゼンタ |
| 緑 + 青 | シアン |
| 赤 + 緑 + 青 | 白 |
